# InfraCode CLI

CLI del Infra AI Framework. Configura agentes de IA en tus proyectos con un solo comando.

## Instalación

```bash
curl -fsSL https://github.com/emanuelarcos-infra/infracode-cli/releases/latest/download/install.sh | bash
```

### Plataformas soportadas

| OS | Arquitectura |
|---|---|
| macOS | Intel (amd64) |
| macOS | Apple Silicon (arm64) |
| Linux | x86-64 (amd64) |
| Linux | ARM64 |

## Uso

```bash
infracode init
```

## Actualización

Para actualizar a la última versión, corré el mismo comando de instalación.
